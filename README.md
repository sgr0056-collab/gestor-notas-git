# Gestor de Notas en Python

## Descripción
Este proyecto es una aplicación de consola sencilla desarrollada en Python diseñada para gestionar notas de texto de forma dinámica. [cite_start]El programa permite al usuario administrar una lista de recordatorios o tareas pendientes mediante una interfaz de menú interactiva. [cite: 48]

## Funcionalidades
[cite_start]Basándose en la lógica implementada en el código, el sistema permite: [cite: 49]
- **Añadir notas:** Permite al usuario ingresar una cadena de texto que se almacena en la sesión actual.
- **Listar notas:** Muestra de forma numerada todas las notas que han sido guardadas hasta el momento.
- **Eliminar notas:** Facilita el borrado de entradas específicas seleccionándolas por su número de índice.
- **Interfaz de menú:** Un bucle continuo que permite navegar entre las opciones de gestión o salir del programa de forma segura.

## Cómo ejecutar el programa
[cite_start]Para poner en marcha el gestor de notas, sigue estos pasos: [cite: 50]
1. Asegúrate de tener instalado **Python 3** en tu equipo.
2. Descarga los archivos del proyecto (`main.py`, `notas.py` y `utilidades.py`) en una misma carpeta.
3. Abre la terminal o línea de comandos en dicha ubicación.
4. Ejecuta el comando:
   ```bash
   python main.py
